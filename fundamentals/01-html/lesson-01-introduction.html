Formatting

Physical Tags

The b tag is used to make the text bold.

The i tag is used to make the text italic.

The small tag is used to make the text smaller.

The u tag is used to underline text.


Logical Tags

The strong tag is used to bold text and give it semantic importance.

The em tag is used to italicize text and give it semantic emphasis.


Other Tags

The sup tag is used to make superscript text (slightly above the normal line).

The sub tag is used to make subscript text (slightly below the normal line).

The ins tag is used to indicate that content has been added (typically underlined).

The del tag is used to indicate that content has been deleted (typically struck through).

The mark tag is used to highlight text with a yellow background.
